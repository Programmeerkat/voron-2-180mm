# Skirts for Voron 2.4 180mm

## Overview
The Voron 2.4 can easily scaled down to 180mm with the original 3d-printed parts, with the exception of the skirts. These skirts use the original display and are compatible with 2.4R2. This mod uses standard Voron parts. 

## BOM
* M3 T-nuts or M3 T-nut HNTAJ5-3 (11 pieces)
* M3x8 SHCS (67 pieces)
* M3x5x4 heatset inserts (56 pieces)

## Printing instructions
The parts can be printed with Voron's default settings. When placed diagonally the minimal required bed size is 22 cm by 22 cm. 

## Assembly instructions
The assembly should be straightforward. Comparing to the original model where the fans only use inserts on one size, this implementation uses heatset inserts on both sides of the fans, and uses clips to attach the fans to the skirts for a sturdier result.

1. Add heatset inserts into both sides of the fans or fan dummy (32 heatset inserts), the side skirt pieces (16 heatset inserts), the front skirt pieces (4 heatset inserts), and the rear skirt piece (4 heatset inserts).
2. Use M3x8 SHCS to mount the belt guards to the front and rear skirt pieces (4 M3x8 SHCS total)
3. Use M3x8 SHCS and M3 T-nuts (11 of each) to mount the skirts to the frame.
